# Capstone Project: AI, NLP & Agents with Microsoft Azure

This is the final milestone of the AI, NLP & Agents with Microsoft Azure learning path. By now you've covered LLM fundamentals, prompt engineering, Azure AI Language, Foundry, agents, RAG, and responsible AI. This project is where all of that comes together into one working thing you actually built.

## Pick One Project

### Option A: RAG Based Q&A Agent
Build an agent that answers questions using your own documents, not just general knowledge.
- Pull together a small document set (PDFs, notes, or any knowledge base you care about)
- Use Azure AI Search for retrieval and Foundry IQ or Foundry Agent Service to generate grounded answers
- Make sure answers cite the source document they came from
- Builds on Milestone 5 (Foundry) and Milestone 7 (RAG)

### Option B: Tool Calling AI Agent
Build an agent that can actually take actions, not just chat back and forth.
- Give it at least two custom tools (something like a calculator, a weather lookup, or a database query)
- Use Microsoft Foundry Agent Service with Agent Framework to orchestrate it
- The agent should decide on its own when a tool is actually needed
- Builds on Milestone 5 (Foundry) and Milestone 6 (Agents)

### Option C: Nepali English NLP Tool
Build something that solves a real low resource language problem.
- Pick one: sentiment analysis, translation, or speech transcription for Nepali or code switched text
- Use Azure AI Language as the core engine
- Test it against a small set of examples you've labeled yourself, and be honest about where it fails
- Builds on Milestone 2 (LLMs) and Milestone 4 (Azure AI Language)

## What to Turn In

- [ ] A working project (repo, notebook, or live demo, whatever fits)
- [ ] A short write up explaining what you built and why you made the choices you did
- [ ] A few real examples showing it works
- [ ] Notes on how you approached prompt design, if that applies to your project
- [ ] A short section on what could go wrong with your system (bias, privacy, failure cases). This ties back to Milestone 8

## How This Gets Evaluated

| What we're looking at | What that means in practice |
|---|---|
| Does it work | Real examples, not just a description of what it's supposed to do |
| Is it grounded | Answers or outputs you can trace back to a source, not made up |
| Responsible AI thinking | You've actually thought about at least one risk or limitation |
| Clarity | Someone else could read your write up and understand what you built |
